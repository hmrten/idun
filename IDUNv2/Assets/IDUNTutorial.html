IDUN Tutorial

Welcome to IDUN tutorial.

Here we will explain basics of using IDUN software and its’ key components.

Main Page

Main Page offers direct access to Sensor Overview. Left side menu start with a Hamburger button . It will allow you to expand Main Menu and see full name for each section assigned to the icon.

Bottom Bar menu will give you access to pages controls and notifications.

Bottom Bars Standard buttons will appear and change depending, if the current page we are visiting, supports them.

 Create New  Save Existing  Clear or Remove

 Notification Pop-up

 Access to the Notifications List and number of recent Notifications.

Notification List offers detailed overview of all present notifications and control over them.

Sensor Overview

Sensor Overview offers graphical representation of the sensor data. It will show current value, if the sensor is operational, offline or faulted, minimum, maximum value as the ceiling and floor threshold values.

By clicking on the sensor you will get access to the sensor settings.

Sensor General Setting will let you configure basic setting of a sensor and it is required for any sensors normal functioning behaviour.

Unit represents in which unit sensor displays data. Range min and max are minimum and maximum range sensor is defined for with technical specifications. Danger low and high represent values that will cause sensor to go to faulted state. Faulted state is responsible for setting off and executing Triggers.

Triggers are defined by Comparer, Value and Template. Comparer will say if the Current Value given by selected Sensor is Below or Above wanted Value. If that condition is met, selected Template will be executed.

Templates are pre-generated fault reports sent directly to IFS cloud.

Every report will contain data which you can see on the image, sensor which was triggered and device information report was sent from.

You can access Fault Reports List:

By selecting certain Fault Report you can get more detailed view for a selected report.

Additional applications will let you access LED Control and Speech Synthesis pages.

LED Control will let you do your own masterwork by letting you draw any pattern on the grid. Selected parts of the grid (colored green) will represent LED lights turned on and colored with a RGB Color Slider Selector color you have chosen previously.

By clicking on a Fill button you will get option to Remove Fill or you can press Remove button on the bottom bar to clear the whole grid. You can also save LED Image as a file or load one you previously saved.

By clicking on a Microphone button on the Bottom Bar you will be directed to Speech Synthesis Page.

This is standard Text-To-Speech control with access to preinstalled voices.

To ensure Fault Reports are sent, device needs to be registered and authorised on a Cloud. For that Device Settings Page is used.

After Valid data has been entered and saved, your device will be able to send fault reports to the Cloud, exchange Templates and see full history of the reports ever made by the device.
